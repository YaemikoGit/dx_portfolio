Sentosa online store
website

Project Brief

The original Sentosa store website is to be re-designed to provide a better user experience for customers using PHP and HTML coding.

Skill

Website design & developement

The challenge

Understanding how to implement certain functions to certain element tags with the CSS styling intact and ensuring website is not complex for user experience.

The result

Able to function well like an online store website. Design was not complex and it fulfils the desire purpose.

Highlights

The website has an “Account” page which lists down each order with date of purchase, order status and the option to cancel order. The list of orders can be sorted out according to alphabetical, date and order status.

The “Cart” page is able to add any item under the “Shop” page. Not only it shows the item you have added, it is able to calculate the price according to the quantity added. There is an “Update Me” option which allows the user to change the quantity and the “Remove” option will remove the entire item from the cart.
